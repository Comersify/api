🌐 Base URLs

Production: https://api.comersify.com

Development: http://localhost:8000

🔐 Authentication & Users

Endpoint Method Description Auth
/signup/ POST Create new account No
/login/ POST Login with credentials No
/refresh-token/ POST Refresh JWT token No
/settings/ GET Get user settings Yes
/settings/update/ POST Update user settings Yes
/store/{id}/ GET Get store details No
/stores/top/ GET Get top stores No

📦 Products & Categories

Endpoint Method Description Auth
/v2/products/ GET List all products No
/v2/products/ POST Create product Vendor
/v2/products/{id}/ GET Get product details No
/v2/products/{id}/ PUT Update product Vendor
/v2/products/{id}/ DELETE Delete product Vendor
/products/super-deals/ GET Get deal products No
/v2/categories/ GET List categories No
/v2/categories/ POST Create category Vendor
/reviews/{id} GET Get product reviews No

🛒 Cart

Endpoint Method Description Auth
/cart/products/ GET Get cart contents Yes
/cart/add-product/ POST Add product to cart Yes
/cart/delete-product/ POST Remove from cart Yes
/cart/update-product/ POST Update quantity Yes
/cart/apply-coupon/ POST Apply coupon Yes

📋 Orders

Endpoint Method Description Auth
/my-orders/ GET Get customer orders Yes
/order/create/ POST Create order from cart Yes
/one-seller/order/create/ POST Direct order Yes
/vendor/orders/ GET Get vendor orders Vendor
/vendor/orders/ PUT Update order status Vendor
/vendor/analytics/orders/ GET Order analytics Vendor

❤️ Wishlist

Endpoint Method Description Auth
/wish-list/products/ GET Get wishlist Yes
/wish-list/has-product/{id}/ GET Check product in wishlist Yes
/wish-list/add-product/ POST Add to wishlist Yes
/wish-list/delete-product/ POST Remove from wishlist Yes
