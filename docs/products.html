Products

GET /v2/products/ Auth

List all products with pagination and filtering

Query Parameters

ParameterTypeDescription
pageintegerPage number for pagination
categoryintegerFilter by category ID
searchstringSearch term
POST /v2/products/ Vendor

Create a new product

Request Body

{ "title": "Premium Headphones", "description": "High-quality wireless headphones", "price": 199.99, "buy_price": 149.99, "in_stock": 100, "category": 1, "attributes": [ { "attribute": 1, "values": [1, 2] } ] }
GET /v2/products/{id}/

Get product details

PUT /v2/products/{id}/ Vendor

Update product

DELETE /v2/products/{id}/ Vendor

Delete product

GET /products/id/{id}/

Get product by ID (alternative)

GET /products/super-deals/

Get products with special deals/discounts
