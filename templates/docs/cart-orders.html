Shopping Cart

All cart endpoints require authentication

GET /cart/products/ Auth

Get current user's shopping cart contents

Response

{ "type": "success", "data": { "orders": [ { "id": 1, "product": { "id": 42, "title": "Premium Headphones", "price": 199.99, "image": "https://..." }, "quantity": 1, "coupon": { "code": "SUMMER20", "value": 20 } } ], "total": 199.99 } }
POST /cart/add-product/ Auth

Add a product to the shopping cart

Request Body

{ "product_id": 42 }

Response

{ "type": "success", "message": "Product added to cart" }
POST /cart/delete-product/ Auth

Remove a product from the cart

Request Body

{ "order_id": 1 }
POST /cart/update-product/ Auth

Update the quantity of a product in the cart

Request Body

{ "order_id": 1, "quantity": 3 }
POST /cart/apply-coupon/ Auth

Apply a coupon code to a product in the cart

Request Body

{ "code": "SUMMER20" }

Customer Orders

GET /my-orders/ Auth

Get list of orders for the authenticated customer

Response

{ "type": "success", "data": [ { "id": 1, "product__title": "Premium Headphones", "quantity": 1, "price": 199.99, "status": "SUBMITTED", "created_at": "2024-01-15T10:30:00Z" } ] }
POST /order/create/ Auth

Create a new order from cart items

Request Body

{ "address": "123 Main St, City, Country", "phoneNumber": "+1234567890", "postalCode": "12345", "shippingID": 1 }

Response

{ "type": "success", "message": "Your order was submitted wait the seller to ship it" }
POST /one-seller/order/create/ Auth

Create a direct order with an individual seller

Request Body

{ "fullName": "John Doe", "quantity": 1, "productID": 42, "address": "123 Main St", "phoneNumber": "+1234567890", "postalCode": "12345", "shippingID": 1 }

Vendor Order Management

Requires vendor authentication

GET /vendor/orders/ Vendor

Get list of orders for the vendor's products

Response

{ "type": "success", "data": [ { "id": 1, "product": { "id": 42, "title": "Premium Headphones", "price": 199.99 }, "quantity": 1, "status": "SUBMITTED", "shipping_info": { "address": "123 Main St", "phone_number": "+1234567890", "postal_code": "12345" }, "created_at": "2024-01-15T10:30:00Z" } ] }
PUT /vendor/orders/ Vendor

Update the status of an order

Request Body

{ "order_id": 1, "status": "SHIPPED" }
GET /vendor/analytics/orders/ Vendor

Get order statistics for vendor analytics

Response

{ "type": "success", "data": { "labels": ["Jan", "Feb", "Mar"], "data": [10, 15, 20] } }

Order Status Values

StatusDescription
IN_CARTItem is in shopping cart
SUBMITTEDOrder has been placed
SHIPPEDOrder has been shipped
DELIVEREDOrder has been delivered

Error Responses

{ "type": "error", "message": "Error description" }

Common Errors

MessageDescription
Data is missingRequired fields not provided
Product does not existInvalid product ID
