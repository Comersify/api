Get Wishlist

GET /wish-list/products/ Auth

Get current user's wishlist items

Response

{ "type": "success", "data": { "wishlists": [ { "id": 1, "product": { "id": 42, "title": "Premium Headphones", "price": 199.99, "image": "https://cdn.comersify.com/products/headphones.jpg", "description": "High-quality wireless headphones" }, "created_at": "2024-01-15T10:30:00Z" } ], "count": 1 } }

Check Product in Wishlist

GET /wish-list/has-product/{id}/ Auth

Check if a product is in the user's wishlist

Response (In Wishlist)

{ "type": "success", "inWishList": true }

Response (Not in Wishlist)

{ "type": "success", "inWishList": false }

Add to Wishlist

POST /wish-list/add-product/ Auth

Add a product to the wishlist

Request Body

{ "product_id": 42 }

Example cURL

curl -X POST https://api.comersify.com/wish-list/add-product/ \ -H "Authorization: Bearer <token>" \ -H "Content-Type: application/json" \ -d ''{"product_id": 42}''

Response

{ "type": "success", "message": "Product added to wishlist" }

Remove from Wishlist

POST /wish-list/delete-product/ Auth

Remove a product from the wishlist

Request Body

{ "product_id": 42 }

Response

{ "type": "success", "message": "Product removed from wishlist" }

Implementation Examples

JavaScript - Add to Wishlist

async function addToWishlist(productId) { const response = await fetch('/wish-list/add-product/', { method: 'POST', headers: { 'Authorization': `Bearer ${accessToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ product_id: productId }) }); const data = await response.json(); if (data.type === 'success') { updateWishlistIcon(true); showNotification('Added to wishlist!'); } }

React Component

const WishlistButton = ({ productId, isInWishlist }) => { const handleClick = async () => { const endpoint = isInWishlist ? '/wish-list/delete-product/' : '/wish-list/add-product/'; await fetch(endpoint, { method: 'POST', headers: { 'Authorization': `Bearer ${token}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ product_id: productId }) }); }; return ( <button onClick={handleClick}> {isInWishlist ? '❤️' : '🤍'} </button> ); };

Error Responses

{ "type": "error", "message": "Error description" }

Common Error Messages

ErrorDescription
Product not foundProduct with given ID doesn't exist
Product already in wishlistProduct was already added
Product not in wishlistTried to remove non-existent item
Authentication requiredNo valid token provided

Notes

  • User-specific - Wishlist is tied to authenticated user
  • No duplicates - Adding same product twice doesn't create duplicates
  • Cascade delete - Removing product from platform also removes from wishlists
  • Real-time sync - Changes reflect immediately across devices
